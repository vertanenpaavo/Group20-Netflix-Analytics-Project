# Netflix Analytics Project
## Introduction
In this repo, our team has analyzed [Netflix 2025:User Behavior Dataset.](https://www.kaggle.com/datasets/sayeeduddin/netflix-2025user-behavior-dataset-210k-records) More spesifically, user behaviour and recommendations. The project is split in multiple jypyter notebooks. There are multiple different approaches for processing the data. To conserve each members input, notebooks are separate, without modification. Project is currently not in executable form.
